# 🏏 StumpVizz Backend

**StumpVizz** is a high-quality, ad-free cricket live-streaming platform. This repository contains the backend part of the application built with Express.js and PostgreSQL.

## 🌟 Features

✅ **API for Match Fixtures:** Serves cricket match fixtures and stream links.  
✅ **WebSockets Integration:** Real-time communication for instant updates.  
✅ **CORS Handling:** Allows secure cross-origin requests.  
✅ **Environment Variables:** Secure configuration for sensitive data.  

## 🛠️ Tech Stack

- **Backend:** Express.js (TypeScript)   
- **Real-time Communication:** WebSockets  
- **Deployment:** Render  

## 🚀 Getting Started

### 🔧 Prerequisites
- Node.js and npm  

### 💻 Local Setup

1. **Clone the Repository:**
```bash
git clone https://github.com/swgtds/stumpvizz-backend.git
cd stumpvizz-backend
```

2. **Install Dependencies:**
```bash
npm install
```

4. **Start the Backend Server:**
```bash
npm run dev
```
The backend will run on `http://localhost:5000`.

## 🛡️ API Endpoints

- `GET /api/men-matches` → Fetch all upcoming mens matches  
- `GET /api/women-matches` → Fetch all upcoming women matches
